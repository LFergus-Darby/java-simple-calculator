# Java Simple Calculator

A basic Java calculator that allows the user to perform:

- Addition
- Subtraction
- Multiplication
- Division

## Concepts Practiced

- User input using Scanner
- Variables and numbers
- Conditional logic (if / else)
- Basic program flow

## Example

```
Enter first number: 9  
Enter second number: 23  

Choose operation:  
1 - Add  
2 - Subtract  
3 - Multiply  
4 - Divide  

1

Result: 32.0
```
